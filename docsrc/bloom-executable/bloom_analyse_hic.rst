===========================
Analyse Hi-C matrices (API)
===========================

The ``bloom`` tool should be accessible from the command line after you installed bloom
(see :ref:`bloom_installation`).
It provides convenient commands for the generation, modification, and analysis of Hi-C data.

.. note::

    The following tutorials assume you have downloaded all the example data files from
    `out GitHub page <https://github.com/vaquerizaslab/bloom/>`_ and ran the sample analysis outlined
    in :ref:`example-bloom-auto`.
